# Practice Notes

One place for a band's practice material. Collect lyrics, chord charts (ChordPro or freeform), audio recordings, images, and PDFs per song; group songs into ordered sets; share songs and sets at clean URLs like `/<owner>/<slug>`.

Working title: **practicenotes**. Django + django-ninja, HTMX/Alpine, Tailwind, SQLite, S3-compatible storage, deployed on Fly.io.

Status: pre-alpha — see [PLAN.md](PLAN.md) for the v1 implementation plan and the [issues](../../issues) for the backlog.
